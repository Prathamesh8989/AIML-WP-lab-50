JavaScript Guide
Introduction Variables Functions Loops Objects
Introduction

JavaScript is a versatile programming language commonly used in web development.

It allows you to add interactivity to web pages and build dynamic content.

console.log('Hello, world!');

JavaScript can run in the browser or on the server via Node.js.

  • Client-side scripting
  • Server-side scripting
Variables

Variables store data values in JavaScript.

You can declare them using var, let, or const.

let age = 25;

Variable names should be descriptive and follow naming conventions.

  • Case-sensitive
  • No reserved words
Functions

Functions are reusable blocks of code that perform a specific task.

They help organize and modularize code.

function greet(name) { return `Hello ${name}`; }

Functions can accept parameters and return values.

  • Function declarations
  • Function expressions
Loops

Loops allow you to execute code multiple times efficiently.

Common loops include for, while, and do...while.

for(let i=0; i<5; i++){ console.log(i); }

Loops can iterate over arrays and objects.

  • Break and continue statements
  • Nested loops
Objects

Objects are collections of key-value pairs.

They allow you to store related data in one entity.

const person = {name: 'Alice', age: 25};

Access properties using dot notation or brackets.

  • Dynamic property access
  • Nested objects
